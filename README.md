# Prokaryotic Annotation Tool

A Python-based command-line tool for prokaryotic genome annotation.

## Description
This tool detects open reading frames (ORFs) from a nucleotide FASTA file and
generates a GFF3 annotation file.

## Usage
python prokaryotic_annotation_tool.py --input sample.fasta --out output

## Files
- prokaryotic_annotation_tool.py
- sample.fasta
- output.gff
